# 飞机大战坦克 — 产品功能规格书

> 版本: 1.0 | 日期: 2026-05-11 | 状态: 待评审

## 1. 产品定位

**一句话**: 面向碎片化娱乐的竖版射击手游原型——玩家操控战机，从空中打击地面坦克部队，击退一波波敌军进攻。

**目标用户**: 20-40岁轻度游戏玩家，单局3-5分钟，单手可操作。

**核心体验**: 爽快射击 + 策略选择（道具/武器） + 波次推进的节奏感。

---

## 2. 游戏世界观与场景

### 2.1 场景设定
- **视角**: 竖版俯视角，玩家飞机在屏幕上方1/3处，敌人在下方出现
- **背景**: 三层视差滚动——最远层星空、中层云层、近层地形（草地/沙漠/雪地随关卡变化）
- **氛围**: 军事风格 + 轻科幻，配色偏暗色调，爆炸和光效作为视觉高光

### 2.2 角色设定
| 角色 | 说明 |
|------|------|
| 玩家战机 | 青色(#4ecdc4)流线型战斗机，带引擎尾焰 |
| 普通坦克 | 红色(#ff6b6b)方形坦克，2HP，直线前进，定时射击 |
| 快速坦克 | 紫色(#ff9ff3)小型坦克，1HP，速度快，蛇形移动 |
| 重型坦克 | 深红(#c0392b)大型坦克，4HP，射击频率高，子弹散射 |
| 防空车 | 橙色(#f39c12)地对空单位，3HP，发射追踪导弹 |
| Boss | 超大型战车，多阶段，见 §3.3 |

## 3. 核心玩法

### 3.1 操作方式
| 操作 | 触摸 | 键盘(调试) |
|------|------|-----------|
| 移动 | 左侧虚拟摇杆 | WASD / 方向键 |
| 射击 | 自动射击（开火即默认）| 自动 |
| 技能 | 右侧炸弹按钮 | Space |
| 暂停 | 右上角暂停按钮 | Esc/P |

**设计决策**: 采用自动射击 + 手动移动，降低操作门槛。炸弹作为唯一手动技能，突出关键时刻。

### 3.2 波次系统 (Wave System)

每局游戏由连续波次组成，节奏如下：

| 波次 | 内容 | 时长 |
|------|------|------|
| Wave 1-3 | 仅普通坦克，数量3-5 | 各30s |
| Wave 4-6 | 普通坦克 + 快速坦克 | 各30s |
| Wave 7-9 | 加入重型坦克，数量增加 | 各35s |
| Wave 10 | **Boss战** | Boss被击毁 |
| Wave 11+ | 混合编队 + 防空车，难度持续递增 | 各35s |
| 每5波 | Boss战 | Boss被击毁 |

**波次内节奏**:
- 前5秒: 少量敌人试探性出现
- 中段: 敌人密度达到峰值
- 末段: 清场间歇，掉落道具概率提升

**波次间**: 3秒过渡，显示 "Wave X" 大字动画，自动回血1点（若不满）。

### 3.3 Boss 设计

| Boss | 出现波次 | HP | 攻击模式 |
|------|---------|-----|---------|
| 铁壁战车 | Wave 10 | 50 | 直线弹幕 + 散射（HP<50%激活）|
| 暴风要塞 | Wave 15 | 80 | 追踪导弹 + 弹幕 + 召唤小怪（HP<30%激活）|
| 终焉堡垒 | Wave 20 | 120 | 全部攻击模式 + 激光扫射（HP<50%激活）|

Boss出现在屏幕下方1/3处，缓慢上下移动，不会移出屏幕。

### 3.4 道具系统

道具从击毁的敌人位置掉落，自动漂浮3秒后消失。

| 道具 | 图标 | 效果 | 持续时间 | 掉落权重 |
|------|------|------|---------|---------|
| 武器升级 | ⬆️ 蓝 | 子弹升级（见§3.5） | 永久至死亡 | 15% |
| 护盾 | 🛡️ 金 | 抵挡1次伤害 | 1次或10s | 10% |
| 炸弹补给 | 💣 红 | 炸弹+1 | 即时 | 10% |
| 回血 | ❤️ 绿 | 生命+1 | 即时 | 8% |
| 分数加成 | ⭐ 黄 | 10秒内得分×2 | 10s | 7% |

**道具掉落规则**: 普通敌人10%概率，快速坦克15%，重型坦克25%，防空车20%。Boss击毁必掉2-3个道具。

### 3.5 武器升级

玩家初始为单发子弹，拾取 ⬆️ 道具依次升级：

| 等级 | 名称 | 效果 |
|------|------|------|
| Lv1 | 单发 | 1条子弹，射速300ms |
| Lv2 | 双发 | 2条平行子弹，射速280ms |
| Lv3 | 三向 | 3条子弹(前+左30°+右30°)，射速260ms |
| Lv4 | 五向 | 5条散射子弹，射速240ms |
| Lv5 | 激光 | 连续激光束，持续伤害，射速不适用 |

死亡后武器降1级（最低Lv1）。

### 3.6 炸弹技能

- 初始携带1枚，最多存3枚
- 使用效果: 全屏闪白 → 所有敌人受10点伤害 → 消灭所有敌方子弹 → 2秒内无敌
- 拾取 💣 道具增加1枚

### 3.7 计分与连击

| 事件 | 分数 |
|------|------|
| 普通坦克击毁 | 100 |
| 快速坦克击毁 | 150 |
| 重型坦克击毁 | 300 |
| 防空车击毁 | 250 |
| Boss击毁 | 2000 × Boss等级 |

**连击系统 (Combo)**:
- 1.5秒内连续击杀维持连击
- 连击倍率: 1-4杀=×1, 5-9杀=×1.5, 10-19杀=×2, 20+杀=×3
- 连击中断时显示最终连击数
- 屏幕上方显示当前连击数和倍率

## 4. 玩家状态

### 4.1 生命值
- 初始3点，最多5点
- 受击扣1点（护盾抵挡则不扣）
- 0点时游戏结束

### 4.2 无敌时间
- 受击后1.5秒无敌，飞机闪烁
- 炸弹使用后2秒无敌

## 5. UI 规格

### 5.1 游戏内HUD
```
┌──────────────────────────┐
│ 🎯 12,500   ❤️❤️❤️   ⏸️ │  ← 顶栏: 分数 | 生命 | 暂停
│     ×2.5 COMBO 18        │  ← 连击显示(战斗时出现)
│                          │
│         [游戏区域]         │
│                          │
│   💣×2                   │  ← 右下: 炸弹按钮+数量
│  🕹️                      │  ← 左下: 摇杆
└──────────────────────────┘
```

### 5.2 开始界面
- 游戏标题 (渐变动画)
- "开始战斗" 主按钮
- "操作指南" 次按钮
- 最高分显示
- 版本号

### 5.3 游戏结束界面
- "战斗结束" 标题
- 本局分数 (大字) + 是否新纪录
- 到达波次
- 最大连击
- 击杀统计
- "再战一次" 按钮
- "返回首页" 按钮

### 5.4 暂停界面
- 半透明遮罩
- 继续 / 重新开始 / 返回首页
- 当前分数和波次

### 5.5 波次过渡动画
- 黑色半透明遮罩
- 大字 "WAVE X" 从中间放大淡出
- 3秒后自动进入下一波

## 7. 音效规格

### 7.1 音效列表

| 事件 | 音效风格 | 时长 |
|------|---------|------|
| 玩家射击 | 短促激光声 | 50ms |
| 敌人射击 | 低沉炮声 | 100ms |
| 爆炸(小) | 中等爆炸 | 300ms |
| 爆炸(Boss) | 大爆炸+回响 | 800ms |
| 道具拾取 | 清脆叮声 | 150ms |
| 武器升级 | 上升音阶 | 300ms |
| 受击 | 金属撞击 | 200ms |
| 炸弹 | 轰鸣+回响 | 1000ms |
| 连击里程碑 | 鼓点 | 100ms |
| 波次开始 | 号角 | 500ms |

### 7.2 BGM
- 战斗BGM: 循环，节奏偏快的电子军事风
- Boss战BGM: 更紧张的变奏

### 7.3 实现方式
原型阶段使用 Web Audio API 程序化合成音效（不依赖外部音频文件），BGM暂用简单循环节奏。

## 9. 技术约束

| 项目 | 要求 |
|------|------|
| 运行环境 | 现代移动浏览器 (Chrome/Safari), 单HTML文件 |

## 10. 原型交付标准

以下功能必须在原型中可体验：

- [ ] 视差滚动三层背景
- [ ] 4种敌人（普通/快速/重型/防空车）各有独立外观和行为
- [ ] 波次系统，Wave 1-3 可完整体验
- [ ] Boss战 (Wave 3, 降低门槛便于体验)
- [ ] 道具系统（5种道具均可拾取生效）
- [ ] 武器升级 (Lv1→Lv3，Lv4-5可选)
- [ ] 炸弹技能
- [ ] 连击系统 + UI显示
- [ ] 波次过渡动画
- [ ] 完整的 开始→战斗→结束→重开 流程
- [ ] 游戏结束统计界面
- [ ] 暂停功能
- [ ] 本地最高分
- [ ] 程序化音效（至少射击+爆炸+拾取）
- [ ] 虚拟摇杆 + 触摸操作
