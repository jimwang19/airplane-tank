## 一、设计系统 (Design Tokens)

### 1.1 颜色

| Token | 值 | 用途 |
|-------|----|------|
| `--bg-deep` | `#0a0a15` | 全局背景 |
| `--overlay-bg` | `rgba(10,10,21,.95)` | 界面遮罩背景 |
| `--accent-cyan` | `#4ecdc4` | 主强调色（分数/标题/按钮） |
| `--accent-teal` | `#44a08d` | 主按钮渐变终点 |
| `--accent-red` | `#ff6b6b` | 敌人/危险/结束标题 |
| `--accent-gold` | `#ffd93d` | 新纪录/星级/连击 |
| `--text-primary` | `#ffffff` | 主文字 |
| `--text-secondary` | `#888888` | 次要说明文字 |
| `--text-muted` | `rgba(255,255,255,.4)` | 版本号/辅助信息 |

### 1.2 字体

| 层级 | 字号 | 字重 | 颜色 | 用途 |
|------|------|------|------|------|
| Display | 30px | bold | 渐变(见下) | 游戏主标题 |
| Title | 26px | bold | `--accent-red` | 结束界面标题 |
| Heading | 22px | bold | `--text-primary` | 暂停/教程标题 |
| Score-Large | 32px | bold | `--accent-cyan` | 结束界面大分数 |
| Body | 13–16px | normal | `--text-secondary` | 说明/统计 |
| Caption | 10–11px | normal | `--text-muted` | 版本号/标签 |

**标题渐变**: `linear-gradient(45deg, #4ecdc4, #44a08d, #ff6b6b)`  
应用于 `-webkit-background-clip: text`，配合 `pulse` 呼吸动画（亮度 1→1.3，2s 循环）

### 1.3 按钮规格

```
┌─────────────────── 主按钮 (btn) ───────────────────┐
│                                                      │
│   width: 75% of overlay, max-width: 200px           │
│   height: ~44px  (padding: 12px 0)                  │
│   border-radius: 30px                               │
│   background: linear-gradient(45deg, #4ecdc4, #44a08d) │
│   font-size: 16px, font-weight: bold                │
│   letter-spacing: 1px, color: #fff                  │
│   :active → scale(0.96)                             │
│                                                      │
└──────────────────────────────────────────────────────┘

┌─────────────────── 次要按钮 (btn-sec) ─────────────┐
│                                                      │
│   同主按钮尺寸                                        │
│   background: rgba(255,255,255,.12)                 │
│   border: 1px solid rgba(255,255,255,.25)           │
│                                                      │
└──────────────────────────────────────────────────────┘

按钮间距: margin: 6px auto
```

### 2.2 元素详细规格

#### 主标题
```
文字:        "飞机大战坦克"
字号:        30px

颜色:        渐变 45deg → #4ecdc4 → #44a08d → #ff6b6b
             (-webkit-background-clip: text)
动画:        pulse (brightness 1→1.3, 2s ease-in-out infinite alternate)

```

#### 副标题
```
文字:        "空地对决 · 波次推进"
字号:        13px
颜色:        #888888

```

#### 功能标签组
```
容器:   display:flex, gap:6px, flex-wrap:wrap, justify-content:center
        margin-bottom: 20px

单个标签 (.stag):

  border:     1px solid rgba(255,255,255,.15)
  border-radius: 20px
```

## 三、游戏结束界面 (game-over)

### 3.1 布局线框图

```
┌─────────────────────────────────────┐  ← 480px
│         rgba(10,10,21,.95)          │
│                                     │
│           ↕ 弹性居中布局             │
│                                     │
│           战斗结束                   │  ← 26px bold #ff6b6b
│                                     │   text-shadow: 0 0 12px rgba(255,107,107,.5)
│          ⭐  ⭐  ⭐                  │  ← 星级评价 28px (1-3颗点亮)
│        (gap:8px, margin:4px 0 14px) │   未点亮: grayscale(1) brightness(.25)
│                                     │   点亮: starPop 动画 (scale 0→1.3→1)
│  ┌──────────────────────────────┐  │
│  │    🏆 新纪录!   (条件显示)    │  │ ← 13px #ffd93d pulse动画
│  └──────────────────────────────┘  │   display:none 时不占位
│                                     │
│           88,850                    │  ← 32px bold #4ecdc4 本局分数
│                                     │   text-shadow: 0 0 12px rgba(78,205,196,.5)
│   到达波次: 7    最大连击: 23        │  ← 13px #aaa line-height:1.8
│   击杀数: 46    累计击杀: 312        │
│               ↕ 16px               │
│  ┌──────────────────────────────┐  │
│  │         再战一次              │  │ ← 主按钮
│  └──────────────────────────────┘  │
│               ↕ 6px                │
│  ┌──────────────────────────────┐  │
│  │         返回首页              │  │ ← 次要按钮
│  └──────────────────────────────┘  │
│                                     │
│     飞机大战坦克 v2.0               │  ← 10px rgba(255,255,255,.25)
│                                     │   margin-top:8px
└─────────────────────────────────────┘  ← 800px
```

### 3.2 元素详细规格

#### 标题
```
文字:   "战斗结束"
字号:   26px
字重:   bold
颜色:   #ff6b6b
阴影:   text-shadow: 0 0 12px rgba(255,107,107,.5)
```

#### 星级评价
```
ID:     #go-stars
图标:   3个 ⭐ 并排  (gap:8px, font-size:28px)

点亮规则:
  ★☆☆ (1星) → 任意完成 (wave >= 1 且死亡)
  ★★☆ (2星) → wave >= 5
  ★★★ (3星) → wave >= 10

未点亮样式:   filter: grayscale(1) brightness(.25)
点亮样式:     filter: none
点亮动画:     @keyframes starPop
              0%   { transform: scale(0) rotate(-20deg) }
              70%  { transform: scale(1.3) }
              100% { transform: scale(1) }
              时间: 0.5s, 逐颗延迟播放（建议 0ms / 200ms / 400ms）
```

#### 新纪录提示
```
ID:      #new-rec
文字:    "🏆 新纪录!"
字号:    13px
颜色:    #ffd93d
动画:    pulse (brightness 1→1.3, 0.6s infinite alternate)
触发:    本局分数 > localStorage 历史最高分
不触发:  display:none (不占布局空间)
```

#### 大分数
```
ID:       #go-score
内容:     本局最终得分（整数，无小数）
字号:     32px
字重:     bold
颜色:     #4ecdc4
阴影:     text-shadow: 0 0 12px rgba(78,205,196,.5)
margin-bottom: 12px
```

#### 统计数据
```
ID:       #go-stats
字号:     13px
颜色:     #aaa (#888888)
行高:     1.8
margin-bottom: 16px

格式（HTML）:
  "到达波次: {wave}  &nbsp;&nbsp;  最大连击: {maxCombo}"
  "击杀数: {kills}  &nbsp;&nbsp;  累计击杀: {totalKills}"

注：totalKills 为累计历史数据（跨局持久化）
```

#### 按钮区
```
"再战一次" → 触发: resetGame() + startGame()
             同一局配置，立即重新开始
"返回首页" → 触发: showScreen('start-screen')
             BGM 停止，重置游戏状态
```

### 3.3 显示时序

```
游戏结束触发 (lives = 0)
    │
    ├─ 1. 记录统计（分数/波次/连击/击杀）
    ├─ 2. 更新 localStorage（highscore / maxwave / totalKills）
    ├─ 3. 停止 BGM
    ├─ 4. 显示 game-over 覆盖层（display:flex）
    ├─ 5. 填入大分数、统计文字
    ├─ 6. 判断新纪录 → 显示/隐藏 #new-rec
    └─ 7. 计算星级 → 逐颗播放 starPop 动画（延迟序列）
```

### 3.4 交互流转

```
┌──────────────────────────────────────┐
│           game-over                  │
│                                      │
│  ┌──────────┐      ┌──────────────┐  │
│  │ 再战一次 │      │  返回首页    │  │
│  └────┬─────┘      └──────┬───────┘  │
└───────┼────────────────────┼──────────┘
        │                    │
        ▼                    ▼
┌──────────────┐    ┌──────────────────┐
│ [Canvas游戏] │    │  start-screen    │
│phase=playing │    │  (重置所有状态)   │
│ (完全重置)   │    └──────────────────┘
└──────────────┘
```

## 五、待确认问题

以下内容规格书尚未明确，实现时需要决策：

| # | 问题 | 当前实现 | 建议 |
|---|------|---------|------|
| 1 | 星级评价的解锁门槛是否合理？ | wave≥5=2星, wave≥10=3星 | 原型阶段可接受，正式版考虑 Boss 击杀作为条件 |
| 2 | 结束界面是否展示"到达最高波次"历史 | 未展示 | 可在累计击杀旁补充 |
| 3 | 新纪录动画是否需要更强烈的视觉反馈？ | 仅文字 pulse | 可考虑短暂撒花粒子效果 |
| 4 | 再战一次是否保留武器等级/炸弹？ | 完全重置 | 建议保持完全重置（公平） |
